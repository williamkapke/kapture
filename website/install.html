Installation

Install the Kapture extension

Kapture needs a Chromium-based browser. Supported browsers include Chrome, Edge, Brave, Opera, and Vivaldi.

Download Chrome Download Edge Download Brave

Install from the Chrome Web Store

The easiest way to install Kapture — no developer mode required.

Add to Chrome

Manual installation

Manual installation requires developer mode and may show occasional warnings about developer-mode extensions.

1

Download the extension

Clone or download the Kapture repository from GitHub to get the extension files.

Download Latest Release View on GitHub

2

Load it into your browser

  1. Open chrome://extensions/ in your browser.
  2. Enable Developer mode with the toggle in the top-right.
  3. Click Load unpacked and select the extracted folder (download) or the /extension folder (cloned repo).
3

Pin the extension

Click the puzzle-piece icon in the toolbar, then the pin next to Kapture MCP Browser Automation.
